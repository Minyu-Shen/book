8.1 机票动态定价

航空公司收益管理（revenue management）是有限时域 MDP 的经典应用场景。航班起飞时刻是不可延迟的硬截止，卖不出去的座位价值归零。在这个例子中，我们将看到最优定价策略如何同时依赖于剩余座位数和剩余时间，并讨论当顾客具有策略性等待行为时，单智能体 MDP 建模的局限性。

问题设定

一个航班还有 $T = 3$ 个销售时段和 $C = 2$ 个座位。每个时段，航空公司选择票价 $p \in \{500, 200\}$（全价票或折扣票）。每个时段有 $\lambda = 0.7$ 的概率来一位潜在顾客，$0.3$ 的概率无人问津。到来的顾客分为两种类型：

  • 商务旅客：时间敏感，愿意支付全价 $500$。
  • 休闲旅客：价格敏感，最多接受折扣价 $200$。

关键假设：越接近起飞，商务旅客的比例越高（临时出差需求增加）。设时段 $k$ 中到来的顾客是商务型的概率为 $\alpha_k$：

时段$k = 3$（28 天前）$k = 2$（14 天前）$k = 1$（起飞前一天）
商务旅客比例 $\alpha_k$$0.1$$0.5$$0.9$
每时段高价卖出概率$\lambda \alpha_k = 0.07$$0.35$$0.63$

MDP 建模

MDP 五要素
  • 状态：$(c, k)$，其中 $c$ 为剩余座位数，$k$ 为剩余时段数。
  • 动作：票价 $p \in \{500, 200\}$。
  • 转移：定价 $p = 500$ 时，仅商务旅客购买，卖出概率 $\lambda \alpha_k$；定价 $p = 200$ 时，两类顾客均购买，卖出概率 $\lambda$。卖出一张则 $c$ 减一。
  • 奖励：卖出一张票的票价收入（$500$ 或 $200$），未卖出则为 $0$。
  • 终止：$k = 0$（航班起飞）或 $c = 0$（座位售罄）。

注意两个动作的不同性质。高价定价是一个赌注：有较大概率卖不出去（特别是在早期时段），但一旦卖出收入很高。低价定价是一个保底：几乎确保卖出，但收入较低。这与 4.2 节设备维护例子的"先投资后收益"结构类似，但方向相反：维护是"现在付出、未来受益"，而高价定价是"现在冒险、当期受益"。

两个动作的期望当期收入为：

$$\mathbb{E}[\text{高价收入}] = 500 \cdot \lambda \alpha_k, \qquad \mathbb{E}[\text{低价收入}] = 200 \cdot \lambda \tag{8.1-1}$$

在 $k = 1$（起飞前一天），高价期望收入 $= 500 \times 0.63 = 315$，远高于低价的 $140$。在 $k = 3$（28 天前），高价期望收入 $= 500 \times 0.07 = 35$，远低于低价的 $140$。但当期收入只是决策的一半，另一半是：卖出这张票后，剩余座位的未来价值。

反向归纳求解

这是一个有限时域 MDP，用反向归纳求解。设 $V^*(c, k)$ 为剩余 $c$ 个座位、$k$ 个时段时的最优期望总收入。边界条件 $V^*(c, 0) = 0$（起飞后座位价值为零）和 $V^*(0, k) = 0$（无座位可卖）。设 $\gamma = 1$（短期问题，不折扣）。

贝尔曼最优方程为：

$$V^*(c, k) = \max\Big\{\underbrace{\lambda \alpha_k \cdot \big[500 + V^*(c{-}1, k{-}1)\big] + (1 - \lambda\alpha_k) \cdot V^*(c, k{-}1)}_{\text{高价}},$$ $$\underbrace{\lambda \cdot \big[200 + V^*(c{-}1, k{-}1)\big] + (1 - \lambda) \cdot V^*(c, k{-}1)}_{\text{低价}}\Big\} \tag{8.1-2}$$

逐步计算（从 $k = 1$ 向前推导）：

$k = 1$（起飞前一天），$\alpha_1 = 0.9$

不管剩余座位是 $1$ 还是 $2$，这是最后的销售机会。

$(c = 1, k = 1)$：$Q(\text{高价}) = 0.63 \times 500 = 315$，$Q(\text{低价}) = 0.7 \times 200 = 140$。高价胜出。

$(c = 2, k = 1)$：同理，$V^*(2, 1) = 315$。最后一天不管剩几个座位，都应该高价。

$k = 2$（14 天前），$\alpha_2 = 0.5$

$(c = 1, k = 2)$：只剩 1 个座位，2 个时段。

  • 高价：$Q = 0.35 \times 500 + 0.35 \times V^*(0,1) + 0.65 \times V^*(1,1) = 175 + 0 + 204.75 = 379.75$
  • 低价：$Q = 0.7 \times 200 + 0.7 \times V^*(0,1) + 0.3 \times V^*(1,1) = 140 + 0 + 94.50 = 234.50$

高价胜出（$379.75 > 234.50$）。只剩 1 个座位时，宁可冒险等商务客。

$(c = 2, k = 2)$：2 个座位，2 个时段。$V^*(1,1) = V^*(2,1) = 315$。

  • 高价：$Q = 175 + 0.35 \times 315 + 0.65 \times 315 = 175 + 315 = 490.00$
  • 低价：$Q = 140 + 0.7 \times 315 + 0.3 \times 315 = 140 + 315 = 455.00$

高价胜出（$490 > 455$）。

$k = 3$（28 天前），$\alpha_3 = 0.1$

$(c = 1, k = 3)$：只有 1 个座位，3 个时段。

  • 高价：$Q = 35 + 0.07 \times 0 + 0.93 \times 379.75 = 35 + 353.17 = 388.17$
  • 低价：$Q = 140 + 0.7 \times 0 + 0.3 \times 379.75 = 140 + 113.93 = 253.93$

高价胜出（$388.17 > 253.93$）。即使商务客概率只有 10%，只要有多余时段来"赌"，1 个座位仍应高价。

$(c = 2, k = 3)$：2 个座位，3 个时段。这是最关键的状态。

  • 高价：$Q = 35 + 0.07 \times 379.75 + 0.93 \times 490.00 = 35 + 26.58 + 455.70 = 517.28$
  • 低价：$Q = 140 + 0.7 \times 379.75 + 0.3 \times 490.00 = 140 + 265.83 + 147.00 = 552.83$

低价胜出（$552.83 > 517.28$）！

用代码验证全部计算：

T, C = 3, 2
p_H, p_L = 500, 200
lam = 0.7
alpha = {3: 0.1, 2: 0.5, 1: 0.9}

V = {}
policy = {}
for c in range(C + 1):
    V[(c, 0)] = 0.0
for k in range(T + 1):
    V[(0, k)] = 0.0

for k in range(1, T + 1):
    sell_H = lam * alpha[k]
    for c in range(1, C + 1):
        q_H = p_H * sell_H + sell_H * V[(c-1,k-1)] + (1-sell_H) * V[(c,k-1)]
        q_L = p_L * lam    + lam    * V[(c-1,k-1)] + (1-lam)    * V[(c,k-1)]
        V[(c, k)] = max(q_H, q_L)
        policy[(c, k)] = '高价' if q_H >= q_L else '低价'
        print(f'(c={c},k={k}) Q(高)={q_H:.2f} Q(低)={q_L:.2f} → {policy[(c,k)]}')

最优定价策略

汇总所有状态的最优动作：

$k = 3$
28 天前		 $k = 2$
14 天前		 $k = 1$
起飞前一天
$c = 2$（满座） 低价 $200$ 高价 $500$ 高价 $500$
$c = 1$（剩 1 座） 高价 $500$ 高价 $500$ 高价 $500$
图 8.1-1　最优定价策略。唯一定低价的状态是 $(c=2, k=3)$：座位充裕且离起飞还远。

这个策略网格揭示了一个经典的收益管理洞察：最优定价同时取决于剩余容量和剩余时间。

  • $(c=2, k=3)$：低价。两个座位要在三个时段内卖出。商务旅客此时仅占 10%，高价定价每时段只有 7% 的卖出概率。如果两个时段都坚持高价却没卖出，到了 $k=1$ 虽然高价能卖，但概率也只有 63%，很可能有座位空飞。不如先以低价锁定一张，再用剩余时段为最后一个座位赌高价。
  • $(c=1, k=3)$：高价。只需卖 1 个座位，有 3 个时段来等。即使每时段只有 7% 的商务客概率，三个时段的累计卖出概率约为 $1 - (1-0.07)^3 \approx 20\%$。期望收入 $500 \times 20\% = 100$，与低价的确定性收入 $200 \times 70\% = 140$ 接近，但这里还有一个因素：如果 $k=3$ 没卖出，$k=2$ 和 $k=1$ 的商务客概率更高（50% 和 90%），后续机会的期望值很高。
  • $k=1$（所有容量）：高价。最后机会，$\alpha_1 = 0.9$，高价期望收入 $315$ 远超低价的 $140$。
容量控制与时间控制的交互

最优定价策略可以理解为两股力量的权衡。时间压力推动降价：时间越少，空座位的机会成本越低，不如降价卖出。容量稀缺推动涨价：座位越少，每个座位越珍贵，值得等待高价买家。在 $(c=2, k=3)$，时间充裕但座位相对过剩（2 个座位，只有 3 个时段×0.7≈2.1 个预期顾客），容量压力小于时间压力，低价成为最优选择。

与时域建模的联系

这个例子与 4.2 节的设备维护问题形成了有趣的呼应：

  • 两者都是有限时域 MDP，最优策略都依赖于剩余时间 $k$。
  • 两者的最优策略都通过反向归纳（从终止条件向前推导）精确求出。
  • 两者的状态都需要包含时间信息：设备维护的状态是 $(s, k)$，机票定价的状态是 $(c, k)$。不包含 $k$ 就无法区分"28 天前的 2 个空座"和"起飞前一天的 2 个空座"，而这两者的最优定价完全不同。

不同之处在于：设备维护的"投资"是确定性的（花 $7$ 一定能修好），而高价定价是一个带风险的赌注（高价不一定卖得出去）。这使得定价问题的最优策略对参数更敏感，也更依赖于容量和时间的联合信息。

讨论：当顾客学会等待

上面的求解隐含了一个重要假设：顾客是短视的（myopic）。每位顾客在到达时段做出购买决策，不会预测未来的票价走势。商务旅客看到 $500$ 就买，休闲旅客看到 $200$ 就买，不存在"等等看后面会不会降价"的行为。

在现实中，部分顾客是策略型的（strategic）。如果航空公司总是在起飞前降价促销，理性的休闲旅客会学着推迟购买。更微妙的是，如果航空公司在早期时段定低价（如上面 $(c=2, k=3)$ 的策略），本来愿意出 $500$ 的商务旅客也可能选择在早期以 $200$ 购票。这相当于"早鸟折扣"被高价值顾客利用，降低了航空公司的收入。

策略型顾客的存在使得定价问题从单智能体 MDP 转变为航空公司与顾客群体之间的博弈。航空公司的定价策略影响顾客的等待行为，而顾客的等待行为反过来影响每个时段的需求分布。这种反馈循环无法在标准 MDP 框架内完整捕捉。

研究前沿   策略型顾客下的动态定价是运营管理领域的活跃研究方向。常见的建模方法包括：将顾客的等待决策建模为一个子博弈，在此博弈中求均衡；或者将顾客的策略行为纳入需求模型（例如让 $\alpha_k$ 本身成为定价历史的函数）。这些方法超出了本书的范围，但值得意识到：当决策环境中存在其他理性主体时，单智能体 MDP 的假设可能不再成立。

小结

  1. 机票动态定价是有限时域 MDP 的经典应用。航班起飞时刻是硬截止，剩余座位在起飞后价值为零。
  2. 最优定价策略同时取决于剩余容量和剩余时间。容量充裕时倾向低价锁定销量，容量紧张时倾向高价等待高价值顾客。
  3. 这个例子再次印证了 4.2 节的结论：有限时域问题的最优策略依赖于时间，$k$ 必须纳入状态。
  4. 当顾客具有策略性等待行为时，问题超出了单智能体 MDP 的范畴，需要考虑博弈论视角。
