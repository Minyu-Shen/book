1.3.2 值函数与贝尔曼方程

在1.3.1中，我们初步认识了值函数——它衡量"一个状态有多好"。本节我们将严格定义值函数，从回报 $G_t$ 出发一步步推导出贝尔曼期望方程，展示其矩阵形式，然后通过一个网约车司机的例子验证这些概念。

状态值函数与动作值函数

状态值函数 $V^\pi(s)$

给定策略 $\pi$，状态值函数定义为从状态 $s$ 出发、遵循策略 $\pi$ 行动所获得的累计折扣回报的期望：

$$V^\pi(s) \;=\; \mathbb{E}_\pi\!\left[\, G_t \;\middle|\; S_t = s \,\right], \quad \text{其中 } G_t = \sum_{k=0}^{\infty} \gamma^k R_{t+k}$$
动作值函数 $Q^\pi(s, a)$

在状态 $s$ 下执行特定动作 $a$，之后再遵循策略 $\pi$，所获得的累计折扣回报的期望：

$$Q^\pi(s, a) \;=\; \mathbb{E}_\pi\!\left[\, G_t \;\middle|\; S_t = s,\, A_t = a \,\right]$$

两者的关系：$V^\pi(s) = \sum_{a \in \mathcal{A}(s)} \pi(a \mid s) \; Q^\pi(s, a)$，即 $V^\pi$ 是 $Q^\pi$ 在策略下的加权平均。

贝尔曼期望方程的推导

值函数的定义涉及无穷求和，看似难以计算。但我们可以利用回报 $G_t$ 的递推结构来简化。

第一步：回报的递推分解

将回报 $G_t$ 拆成"当前奖励 + 折扣后的未来回报"：

$$G_t = R_t + \gamma R_{t+1} + \gamma^2 R_{t+2} + \cdots = R_t + \gamma \underbrace{(R_{t+1} + \gamma R_{t+2} + \cdots)}_{G_{t+1}} = R_t + \gamma G_{t+1}$$
第二步：对两端取条件期望

在 $S_t = s$ 的条件下取期望：

$$V^\pi(s) = \mathbb{E}_\pi[G_t \mid S_t = s] = \mathbb{E}_\pi[R_t + \gamma G_{t+1} \mid S_t = s]$$

利用期望的线性性，拆为两项：

$$= \mathbb{E}_\pi[R_t \mid S_t = s] + \gamma \, \mathbb{E}_\pi[G_{t+1} \mid S_t = s]$$
第三步：展开第一项——即时奖励的期望

即时奖励 $R_t = R(S_t, A_t)$ 取决于状态和动作。在策略 $\pi$ 下，动作 $A_t$ 以概率 $\pi(a \mid s)$ 被选取：

$$\mathbb{E}_\pi[R_t \mid S_t = s] = \sum_{a \in \mathcal{A}(s)} \pi(a \mid s) \, R(s, a)$$
第四步：展开第二项——未来回报的期望

$G_{t+1}$ 从 $S_{t+1}$ 开始，利用全期望公式，先对 $A_t$ 和 $S_{t+1}$ 条件化：

$$\mathbb{E}_\pi[G_{t+1} \mid S_t = s] = \sum_{a \in \mathcal{A}(s)} \pi(a \mid s) \sum_{s' \in \mathcal{S}} P(s' \mid s, a) \, \mathbb{E}_\pi[G_{t+1} \mid S_{t+1} = s']$$

注意到 $\mathbb{E}_\pi[G_{t+1} \mid S_{t+1} = s']$ 正是 $V^\pi(s')$（马尔可夫性质保证了这一点）：

$$= \sum_{a \in \mathcal{A}(s)} \pi(a \mid s) \sum_{s' \in \mathcal{S}} P(s' \mid s, a) \, V^\pi(s')$$
第五步：合并，得到贝尔曼期望方程

将第三步和第四步合并：

$$\boxed{V^\pi(s) = \sum_{a \in \mathcal{A}(s)} \pi(a \mid s) \left[\, R(s, a) + \gamma \sum_{s' \in \mathcal{S}} P(s' \mid s, a) \, V^\pi(s') \,\right]}$$

对于确定性策略 $a = \pi(s)$，方程进一步简化为：

$$V^\pi(s) = R\big(s, \pi(s)\big) + \gamma \sum_{s' \in \mathcal{S}} P\big(s' \mid s, \pi(s)\big) \, V^\pi(s')$$

矩阵形式：策略评估 = 求解线性方程组

对于有 $n = |\mathcal{S}|$ 个状态的MDP，贝尔曼期望方程可以写成矩阵形式。给定一个确定性策略 $\pi$，定义：

  • $\mathbf{V} \in \mathbb{R}^n$：值函数向量，$\mathbf{V}_i = V^\pi(s_i)$
  • $\mathbf{R}_\pi \in \mathbb{R}^n$：奖励向量，$(\mathbf{R}_\pi)_i = R(s_i, \pi(s_i))$
  • $\mathbf{P}_\pi \in \mathbb{R}^{n \times n}$：策略下的转移矩阵，$(\mathbf{P}_\pi)_{ij} = P(s_j \mid s_i, \pi(s_i))$

则贝尔曼期望方程变为：

$$\mathbf{V} = \mathbf{R}_\pi + \gamma \, \mathbf{P}_\pi \, \mathbf{V}$$

移项整理：

$$(\mathbf{I} - \gamma \mathbf{P}_\pi) \, \mathbf{V} = \mathbf{R}_\pi$$

当 $\gamma < 1$ 时，$(\mathbf{I} - \gamma \mathbf{P}_\pi)$ 是可逆的，因此值函数有唯一的解析解：

$$\mathbf{V} = (\mathbf{I} - \gamma \mathbf{P}_\pi)^{-1} \, \mathbf{R}_\pi$$
核心认识：评估一个给定策略 $\pi$ 的好坏，本质上就是求解一个线性方程组，或等价地，求一个矩阵的逆。对于 $n$ 个状态的MDP，计算复杂度为 $O(n^3)$。当状态空间很大时（现实管理问题中常见），直接求逆不可行，需要使用迭代方法——这将在1.4节中介绍。

例子：网约车司机的接单决策

一位网约车司机在一座城市中运营，城市分为三个区域：市中心（$c$）、郊区（$s$）和机场（$a$）。每完成一单后，司机需要决定下一步怎么做。

MDP建模

状态空间：$\mathcal{S} = \{c, s, a\}$，即司机当前所在区域。

动作空间：

  • 市中心（$c$）：接单 或 空驶去机场（赌一把机场大单）
  • 郊区（$s$）：接单 或 空驶去市中心（逃离低收入区域）
  • 机场（$a$）：接单 或 空驶去市中心（放弃等待，主动回城）

转移概率与奖励如下图所示：

状态转移图（三区域网约车MDP）
蓝色实线 = 接单（乘客目的地随机）   橙色虚线 = 空驶（确定性转移，100%）   $\gamma = 0.9$

这个MDP的关键特征：

  • 市中心：收入适中（$R=8$），是"稳定区"——50%留在市中心，且有20%概率自然到达机场
  • 郊区：收入很低（$R=1$），是"陷阱区"——80%概率停留在郊区内循环
  • 机场：收入最高（$R=14$），但每次接单后一定离开（60%回市中心，40%去郊区），是"高回报但不可停留"的区域

决策困境不止一个：在郊区要不要空驶去市中心？在市中心值不值得空驶去机场赌大单？

策略评估：用贝尔曼方程计算 $V^\pi$

策略 $\pi_1$：有单就接

$\pi_1(c) = \text{接单}$
$\pi_1(s) = \text{接单}$
$\pi_1(a) = \text{接单}$

来什么单接什么单，从不空驶。

策略 $\pi_2$：郊区逃离

$\pi_2(c) = \text{接单}$
$\pi_2(s) = \text{空驶去市中心}$
$\pi_2(a) = \text{接单}$

郊区单子太差，花油钱逃回市中心。

求解 $V^{\pi_1}$（有单就接）

三个状态都选择接单，代入贝尔曼方程得三元线性方程组：

$$\begin{cases} V(c) = 8 + 0.9\big[0.5 V(c) + 0.3 V(s) + 0.2 V(a)\big] \\[6pt] V(s) = 1 + 0.9\big[0.1 V(c) + 0.8 V(s) + 0.1 V(a)\big] \\[6pt] V(a) = 14 + 0.9\big[0.6 V(c) + 0.4 V(s)\big] \end{cases}$$

用矩阵形式表示：$(\mathbf{I} - \gamma \mathbf{P}_{\pi_1}) \mathbf{V} = \mathbf{R}_{\pi_1}$，其中

$$\mathbf{P}_{\pi_1} = \begin{pmatrix} 0.5 & 0.3 & 0.2 \\ 0.1 & 0.8 & 0.1 \\ 0.6 & 0.4 & 0 \end{pmatrix}, \quad \mathbf{R}_{\pi_1} = \begin{pmatrix} 8 \\ 1 \\ 14 \end{pmatrix}$$

求解 $\mathbf{V} = (\mathbf{I} - 0.9 \, \mathbf{P}_{\pi_1})^{-1} \mathbf{R}_{\pi_1}$，得到：

$$V^{\pi_1}(c) \approx 51.2, \quad V^{\pi_1}(s) \approx 37.8, \quad V^{\pi_1}(a) \approx 55.3$$

求解 $V^{\pi_2}$（郊区逃离）

$\pi_2$ 在郊区选择空驶（确定性转移到市中心，$R=-3$），其余接单：

$$\begin{cases} V(c) = 8 + 0.9\big[0.5 V(c) + 0.3 V(s) + 0.2 V(a)\big] \\[6pt] V(s) = -3 + 0.9 \, V(c) \\[6pt] V(a) = 14 + 0.9\big[0.6 V(c) + 0.4 V(s)\big] \end{cases}$$

注意第二个方程变得非常简单——空驶是确定性的。对应的转移矩阵变为：

$$\mathbf{P}_{\pi_2} = \begin{pmatrix} 0.5 & 0.3 & 0.2 \\ 1.0 & 0 & 0 \\ 0.6 & 0.4 & 0 \end{pmatrix}, \quad \mathbf{R}_{\pi_2} = \begin{pmatrix} 8 \\ -3 \\ 14 \end{pmatrix}$$

求解得：

$$V^{\pi_2}(c) \approx 62.8, \quad V^{\pi_2}(s) \approx 53.5, \quad V^{\pi_2}(a) \approx 67.2$$

对比与讨论

状态$V^{\pi_1}$（有单就接）$V^{\pi_2}$（郊区逃离）提升
$c$（市中心）51.262.8+11.6
$s$（郊区）37.853.5+15.7
$a$（机场）55.367.2+11.9

$\pi_2$ 在所有状态上大幅优于 $\pi_1$。关键洞察：郊区是一个"陷阱"——接单只赚1元，且80%概率继续留在郊区。花3元油钱逃回市中心，短期亏损，但长期获得的高收入（市中心$R=8$，还有20%概率自然到达机场赚$R=14$）远远弥补了空驶成本。

注意机场并不是"最优目的地"：虽然机场单价最高，但我们不必刻意空驶去机场——在市中心接单就有20%概率自然到达机场。

用 $Q^\pi$ 发现策略的改进方向

在 $\pi_1$（有单就接）下，计算郊区的动作值函数：

$$Q^{\pi_1}(s, \text{接单}) = 1 + 0.9(0.1 \times 51.2 + 0.8 \times 37.8 + 0.1 \times 55.3) \approx 37.8$$
$$Q^{\pi_1}(s, \text{空驶}) = -3 + 0.9 \times 51.2 \approx 43.1$$

$Q^{\pi_1}(s, \text{空驶}) = 43.1 > Q^{\pi_1}(s, \text{接单}) = 37.8$。即使在"有单就接"的策略下评估，空驶的长期回报也更高。这正是改进策略的信号——把郊区的动作从"接单"换成"空驶"，就得到了更好的 $\pi_2$。

再看市中心：是否值得空驶去机场？

$$Q^{\pi_2}(c, \text{接单}) = 8 + 0.9(0.5 \times 62.8 + 0.3 \times 53.5 + 0.2 \times 67.2) \approx 62.8$$
$$Q^{\pi_2}(c, \text{空驶去机场}) = -4 + 0.9 \times 67.2 \approx 56.5$$

$Q^{\pi_2}(c, \text{接单}) = 62.8 > Q^{\pi_2}(c, \text{空驶去机场}) = 56.5$。在市中心接单已经很好了（有自然20%概率去机场），不值得花4元油钱专门跑一趟。

关键认识：贝尔曼期望方程是评估给定策略的工具——它告诉我们"按这个策略走，每个状态值多少"，本质是求矩阵的逆。$Q^\pi$ 进一步揭示"哪些动作比当前策略更好"。如何系统地利用这些信息找到最优策略，是1.4节的主题。

小结

本节完成了四件事：

  1. 从回报 $G_t$ 的递推结构出发，逐步推导了贝尔曼期望方程
  2. 展示了贝尔曼方程的矩阵形式 $\mathbf{V} = (\mathbf{I} - \gamma \mathbf{P}_\pi)^{-1} \mathbf{R}_\pi$，阐明策略评估的本质是求解线性方程组
  3. 通过网约车司机的三区域例子，数值验证了不同策略的值函数差异
  4. 利用 $Q^\pi$ 分析了策略的改进方向，解释了"为什么不必专门跑机场"
