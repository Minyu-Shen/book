1.4.1 策略迭代

上一节中，我们学会了评估一个给定策略 $\pi$ 的好坏：通过求解贝尔曼期望方程，得到每个状态的值函数 $V^\pi(s)$。在网约车例子中，我们评估了两个策略，发现 $\pi_2$（郊区逃离）在所有状态上都优于 $\pi_1$（有单就接）。

但这引出了一个关键问题：$\pi_2$ 是怎么来的？能否从 $\pi_1$ 出发，通过某种系统的方法自动找到更好的策略？本节将回答这个问题。

从策略评估到策略改进

假设我们已经评估了策略 $\pi_1$（有单就接），得到了值函数 $V^{\pi_1}(c)=52.3$，$V^{\pi_1}(b)=42.3$，$V^{\pi_1}(a)=55.9$。现在我们想利用这些信息来改进策略。

改进的思路很直接：在每个状态 $s$，与其盲目按 $\pi_1$ 行动，不如逐一考察每个可选动作，计算如果执行该动作然后再遵循 $\pi_1$，长期回报会是多少。这就是动作值函数 $Q^{\pi}(s,a)$ 的含义，它可以通过 $V^{\pi}$ 和已知的模型直接计算：

$$Q^\pi(s, a) = R(s, a) + \gamma \sum_{s' \in \mathcal{S}} P(s' \mid s, a) \, V^\pi(s') \tag{1.4-1}$$

公式(1.4-1)的含义是：在状态 $s$ 执行动作 $a$，获得即时奖励 $R(s,a)$，然后到达下一个状态 $s'$（按转移概率 $P$），从 $s'$ 开始遵循策略 $\pi$ 的长期回报是 $V^\pi(s')$。这就是一步前瞻（one-step lookahead）的思想。

让我们用网约车的例子来算一算。以 $\pi_1$ 的值函数 $V^{\pi_1}$ 代入公式(1.4-1)，计算每个状态下所有可选动作的 $Q$ 值：

状态动作$Q^{\pi_1}(s,a)$ 的计算$Q^{\pi_1}$
$c$接单$7.3 + 0.9 \times (0.5 \times 52.3 + 0.3 \times 42.3 + 0.2 \times 55.9)$52.3
$c$空驶去机场$-4 + 0.9 \times 55.9$46.3
$b$接单$2.1 + 0.9 \times (0.1 \times 52.3 + 0.8 \times 42.3 + 0.1 \times 55.9)$42.3
$b$空驶去市中心$-3 + 0.9 \times 52.3$44.1
$a$接单$12.4 + 0.9 \times (0.6 \times 52.3 + 0.4 \times 42.3)$55.9
$a$空驶去市中心$-5 + 0.9 \times 52.3$42.1

仔细观察表中黄色高亮的两行：在郊区状态 $b$，空驶去市中心的 $Q$ 值（44.1）高于接单（42.3）。这意味着，如果司机在郊区改为空驶去市中心，然后之后再照常遵循 $\pi_1$，长期回报就会提高 $44.1 - 42.3 = 1.8$。而在市中心和机场，$\pi_1$ 的选择（接单）已经是 $Q$ 值最大的动作，无需改动。

这自然引出了贪婪策略改进的思想：在每个状态，选择使 $Q^\pi(s,a)$ 最大的动作，得到新策略 $\pi'$：

$$\pi'(s) = \arg\max_{a \in \mathcal{A}(s)} Q^\pi(s, a) \tag{1.4-2}$$

将这个规则应用到 $\pi_1$ 上，得到的新策略恰好就是 $\pi_2$（郊区逃离）：市中心和机场继续接单，郊区改为空驶去市中心。1.3.3节中我们已经验证了 $\pi_2$ 在所有状态上优于 $\pi_1$。这不是巧合，而是贪婪改进的一般性质。

策略改进定理

策略改进定理（Policy Improvement Theorem）

设 $\pi$ 是任意策略，$\pi'$ 是基于 $V^\pi$ 的贪婪策略（由公式(1.4-2)定义）。则对所有状态 $s \in \mathcal{S}$：

$$V^{\pi'}(s) \geq V^\pi(s) \tag{1.4-3}$$

即贪婪改进后的策略至少和原策略一样好，且在所有状态上同时成立。

直觉上，贪婪改进保证了每个状态的单步决策都不变差，而这些局部优势经过递推累积起来，就保证了整个策略的全局提升。严格证明见本节末尾的选读部分。

贝尔曼最优方程

如果贪婪改进无法再提升策略，即对所有状态 $s$ 都有 $V^\pi(s) = \max_a Q^\pi(s,a)$，那么 $\pi$ 就已经是最优策略。此时值函数满足一个特殊的方程：

贝尔曼最优方程（Bellman Optimality Equation）
$$V^*(s) = \max_{a \in \mathcal{A}(s)} \left[\, R(s, a) + \gamma \sum_{s' \in \mathcal{S}} P(s' \mid s, a) \, V^*(s') \,\right] \tag{1.4-4}$$

最优策略为：$\pi^*(s) = \arg\max_{a \in \mathcal{A}(s)} \left[\, R(s, a) + \gamma \sum_{s'} P(s' \mid s, a) \, V^*(s') \,\right]$

对比1.3.3节的贝尔曼期望方程(1.3-8)和上面的贝尔曼最优方程(1.4-4)：期望方程对动作做加权平均（按策略 $\pi$ 的概率），最优方程对动作取最大值。前者用于评估一个给定策略，后者刻画最优策略必须满足的条件。

值得注意的是，正是这个 $\max$ 算子使得贝尔曼最优方程成为一个非线性方程组，无法像贝尔曼期望方程那样通过解线性方程组直接求得 $V^*$。这也是为什么我们需要策略迭代、值迭代等迭代算法来逼近最优解。

策略迭代算法

把策略评估和策略改进交替进行，就得到了策略迭代（Policy Iteration）算法：

算法：策略迭代（Policy Iteration）

输入：MDP $\langle \mathcal{S}, \mathcal{A}, P, R, \gamma \rangle$

  1. 初始化：选择任意策略 $\pi_0$（如随机策略或全部选第一个动作）
  2. 重复以下步骤，直到策略不再改变：
    1. 策略评估：求解 $\mathbf{V}^\pi = (\mathbf{I} - \gamma \mathbf{P}_\pi)^{-1} \mathbf{R}_\pi$，得到 $V^\pi(s)$
    2. 策略改进：对每个状态 $s$，令 $\pi'(s) = \arg\max_a \left[ R(s,a) + \gamma \sum_{s'} P(s'|s,a) V^\pi(s') \right]$
    3. 若 $\pi' = \pi$，输出 $\pi^* = \pi$，终止；否则令 $\pi \leftarrow \pi'$，回到步骤 2a

由于有限MDP的确定性策略数量是有限的（$\prod_s |\mathcal{A}(s)|$），且每次改进都严格提升值函数（除非已最优），算法必在有限步内收敛。

例子：网约车司机的策略迭代

让我们在网约车例子上完整运行策略迭代算法。

第一轮迭代

评估 $\pi_1$（有单就接）：由1.3.3节可知，$V^{\pi_1}(c)=52.3$，$V^{\pi_1}(b)=42.3$，$V^{\pi_1}(a)=55.9$。

改进：计算所有状态-动作对的 $Q^{\pi_1}$ 值（见前面的表格），在每个状态取最大的动作：

  • 市中心 $c$：$Q(c,\text{接单})=52.3 > Q(c,\text{空驶去机场})=46.3$，选接单（不变）
  • 郊区 $b$：$Q(b,\text{空驶去市中心})=44.1 > Q(b,\text{接单})=42.3$，改为空驶去市中心
  • 机场 $a$：$Q(a,\text{接单})=55.9 > Q(a,\text{空驶去市中心})=42.1$，选接单（不变）

新策略 $\pi_2 = \{c:\text{接单},\; b:\text{空驶去市中心},\; a:\text{接单}\}$，与 $\pi_1$ 不同，继续迭代。

第二轮迭代

评估 $\pi_2$（郊区逃离）：由1.3.3节可知，$V^{\pi_2}(c)=56.3$，$V^{\pi_2}(b)=47.7$，$V^{\pi_2}(a)=60.0$。

改进：计算 $Q^{\pi_2}$ 值：

状态动作$Q^{\pi_2}(s,a)$贪婪选择
$c$接单$\mathbf{56.3}$接单 ✓
$c$空驶去机场$50.0$
$b$接单$46.9$空驶去市中心 ✓
$b$空驶去市中心$\mathbf{47.7}$
$a$接单$\mathbf{60.0}$接单 ✓
$a$空驶去市中心$45.7$

贪婪策略 $\pi' = \pi_2$，策略没有改变，算法收敛。

策略迭代仅用 2 轮 就从 $\pi_1$ 收敛到最优策略 $\pi^* = \pi_2$。

$$V^*(c) = 56.3, \quad V^*(b) = 47.7, \quad V^*(a) = 60.0$$

可以验证最优值函数满足贝尔曼最优方程(1.4-4)：在每个状态，$V^*$ 恰好等于最大 $Q$ 值。例如在郊区，$V^*(b) = 47.7 = \max(46.9, 47.7) = Q^*(b, \text{空驶去市中心})$。

洞察：策略迭代自动发现了最优策略的关键特征：在郊区不要接单（期望车费仅2.1元，且80%概率留在郊区），应该花3元油钱逃回市中心。这种短期牺牲换取长期收益的权衡，正是值函数所捕捉的。
选读：策略改进定理的严格证明

要证明：对所有状态 $s$，贪婪改进后的策略 $\pi'$ 满足 $V^{\pi'}(s) \geq V^\pi(s)$。

贪婪策略在每个状态选使 $Q^\pi$ 最大的动作，因此一定不差于 $\pi$ 选的动作。将 $Q^\pi$ 按定义展开，并写成条件期望的形式：

$$\begin{aligned} V^\pi(s) &= Q^\pi(s, \pi(s)) \\[4pt] &\leq \max_a Q^\pi(s,a) = Q^\pi(s, \pi'(s)) \\[4pt] &= R(s, \pi'(s)) + \gamma \sum_{s'} P(s' \mid s, \pi'(s)) \, V^\pi(s') \\[4pt] &= \mathbb{E}\!\left[R_{t+1} + \gamma V^\pi(S_{t+1}) \;\middle|\; S_t = s,\, A_t = \pi'(s)\right] \end{aligned} \tag{1.4-5}$$

对所有 $s'$ 都有 $V^\pi(s') \leq Q^\pi(s', \pi'(s'))$，因此可以在期望内部将 $V^\pi(S_{t+1})$ 替换为 $Q^\pi(S_{t+1}, \pi'(S_{t+1}))$，再将这个 $Q^\pi$ 展开为条件期望：

$$\begin{aligned} &\leq \mathbb{E}\!\left[R_{t+1} + \gamma \, Q^\pi\!\big(S_{t+1},\, \pi'(S_{t+1})\big) \;\middle|\; S_t = s,\, A_t = \pi'(s)\right] \\[6pt] &= \mathbb{E}\!\left[R_{t+1} + \gamma \, \mathbb{E}\!\left[R_{t+2} + \gamma V^\pi(S_{t+2}) \;\middle|\; S_{t+1},\, A_{t+1} = \pi'(S_{t+1})\right] \;\middle|\; S_t = s,\, A_t = \pi'(s)\right] \end{aligned}$$

现在出现了嵌套的条件期望。MDP 的马尔可夫性质保证：给定 $(S_{t+1}, A_{t+1})$，未来的 $(R_{t+2}, S_{t+2})$ 与过去的 $(S_t, A_t)$ 条件独立。因此可以在内层条件中加入 $S_t = s, A_t = \pi'(s)$ 而不改变结果。此时内层条件包含了外层条件，可以应用重期望公式（law of iterated expectations）将两层期望合并为一层：

$$\begin{aligned} &= \mathbb{E}\!\left[R_{t+1} + \gamma \, \mathbb{E}\!\left[R_{t+2} + \gamma V^\pi(S_{t+2}) \;\middle|\; S_{t+1}, A_{t+1}, S_t\!=\!s, A_t\!=\!\pi'(s)\right] \;\middle|\; S_t = s,\, A_t = \pi'(s)\right] \\[6pt] &= \mathbb{E}\!\left[R_{t+1} + \gamma R_{t+2} + \gamma^2 V^\pi(S_{t+2}) \;\middle|\; S_t = s,\, A_t = \pi'(s)\right] \end{aligned} \tag{1.4-6}$$

公式(1.4-6)中，前两步的动作都遵循 $\pi'$。对 $V^\pi(S_{t+2})$ 再次执行同样的操作（贪婪不等式 → 展开 $Q$ → 马尔可夫性质 + 重期望公式），不断延长"按 $\pi'$ 行动"的步数：

$$\begin{aligned} &\leq \mathbb{E}\!\left[R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \gamma^3 V^\pi(S_{t+3}) \;\middle|\; S_t = s,\, A_t = \pi'(s)\right] \\[4pt] &\;\;\vdots \\[4pt] &\leq \mathbb{E}\!\left[\sum_{i=0}^{k-1} \gamma^i R_{t+i+1} \;+\; \gamma^k V^\pi(S_{t+k}) \;\middle|\; S_t = s,\, A_t = \pi'(s)\right] \end{aligned}$$

由于 $\gamma < 1$ 且 $V^\pi$ 有界，残余项 $\gamma^k V^\pi(S_{t+k}) \to 0$。令 $k \to \infty$，所有动作都按 $\pi'$ 选取：

$$V^\pi(s) \leq \mathbb{E}_{\pi'}\!\left[\sum_{i=0}^{\infty} \gamma^i R_{t+i+1} \;\middle|\; S_t = s\right] = V^{\pi'}(s) \tag{1.4-7}$$

对所有状态 $s$ 成立。证毕。

策略迭代何时终止

当 $\pi' = \pi$ 时，公式(1.4-5)的第二行不等号变为等号，即 $V^\pi(s) = \max_a Q^\pi(s,a)$，这恰好是贝尔曼最优方程(1.4-4)。因此 $\pi$ 已是最优策略。有限 MDP 的确定性策略数量为 $\prod_s |\mathcal{A}(s)|$，每次改进都严格提升值函数（除非已最优），因此策略迭代必在有限步内收敛。
